# IMFlask-boilerplate
Boilerplate for Large Scale Flask Web Backend Structure



# Get Started

```shell
$ pip install -r ./requirements/requirements.txt
$ export FLASK_APP=manage:application
$ export FLASK_ENV=development
$ flask run
```
